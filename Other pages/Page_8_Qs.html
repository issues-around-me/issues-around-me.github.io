What is a wireframe for a website? Why is it necessary before starting to code up an actual website?

The wireframe is a layout either represented in drawing or graphic form that shows the layout of a website.
It is essential to the website construction process because it represents what you want in the website and how you want it to look.

What are HEX (hexadecimal) color codes? Give one example of a color and its HEX color code.

HEX colors (example: #F39FFF (Mauve - This box)) are colors formatted as triplets of code (# | F3 | 9F | FF), each ranging from 0 to FF (letters A to F represent numbers higher than 10 that fit into one digit space.) They are used to represent colors throughout the color spectrum, and are alternatives to RGB and CMYK, and are more precise than simple color names.

Give example of a few subpages that the navigation of a website should have. What purpose do these subpages serve?

Subpages, such as this one, are used to include more content into a website so that all of it isn't simply on one page, which is very cluttered.
Some important subpages could be a sources page (for research) or simply "about" sections.

What is a website (graphical) mock-up? How does it differ from a wireframe?

A website mock-up is actually putting the wireframe into digital format, and putting it into an html format, while a wireframe is simply the layout.

What is the difference between HTML and CSS?

HTML is the structure and shape of a website, which includes the content.
CSS is the look and position of these elements.
HTML and CSS are often used alongside each other to make an aesthetically pleasing website.
