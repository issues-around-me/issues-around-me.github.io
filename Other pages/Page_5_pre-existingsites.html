My first website

I use this website very often due to the fact that i simply adore Lord of the Rings, and this provides countless resources that i can use, including some to teach me Elvish.

Please use your scroll button on your mouse to scroll through the page. Yes, it is navigable!

This website is very well organized, with tons of pages opening up to hundreds of other links.
Even though there is a "login" function, that is just for the forum section.
The website is well designed, with a very appealing layout which is almost directly taken from the books the website is based on. Also, when you refresh the page, there is a new image of a character from the LOTR universe, which is also a very appealing function.
The website is extremely useful, in which there are countless resources, like bookmarks, icons for your desktop, resources to learn Elvish, fonts, and other things from the LOTL universe.
The website is very easy to use, but one can easily get lost in it, or lose a certain resource, which can be a pain. This has happened to me before, in which I had lost the link to a Sindarin (an elvish dialect) dictionary.
The website is not the most modern website I've seen, and most of it looks like it was built in the 90s or in the early 2000s, possibly with HTML 3 or 4. It also looks like no one has been updating the website in a while, as stated above, the format looks like HTML 3 or 4, possibly with CSS2.
The website is perfect for its users (fans of Lord of the Rings), and allows them to access countless resources and chat with other fans if they are logged in.

 

My second website

This is a website I found through random Internet searches, and it caught my eye.

This website is very minimalist, and is probably built using Bootstrap (a form of CSS used to make designing a page easier) It is very easy to look at, and offers a very nice mix of colors and different shades of those colors. There are also many animations throughout the page, which are very pleasing to the eye. The website (and specifically the Animations) work through a series of div elements, as seen below:

Here, you can see that each section has a certain div element, and if you expand each one:

This would have taken a very long time to build, and the use of Bootstrap is evident.
The website uses the information it provides as a simple way to advertise a program about how to focus more at work, and uses a timer (probably made with JavaScript) and works very well, considering that people most likely went on it as a form of distraction. The link at the bottom of the page takes you to a software advertising its ability to improve team organization and such (Click here to see).
The website, when opened in its native form, is a bit difficult to read, since only a small amount of the page is shown at a time. This is not very ergonomic, and puts readers with reading disabilities at a large disadvantage.
It is very modern-looking, and although it is difficult to find if you are not looking for it, The website is exemplary for its audience, which is most likely anyone in a workplace with access to the Internet. It provides useful, statistical, and trustworthy information that proves a good point (people waste time in the workplace).
